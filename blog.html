Question-1: What are the different ways to select an element in the DOM?
The getElementById("id") indicate single element in unique id . The getElementsByClassName("class") indicate all same class in html file, getElementsByTagName("tag") indicate tag name in inner Text.querySelector("selector") selects the first matching element.querySelectorAll("selector") select NodeList of all matching elements
Question-2: What is the difference between innerHTML, innerText, and textContent ?
InnerHTML returns or sets the HTML including tags and child elements. innerText return or sets the visible text element.textContent return or sets the full text content inside an element, including hidden elements.
Question-3: What is event delegation in the DOM?
Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern ,handle events at a higher level in the DOM tree other than the level where the event was first received.
Question-4: What is event bubbling in the DOM?
Event bubbling is a mechanism where an event triggered on a child element propagates in the DOM. It allows parent elements to respond to events triggered by their child elements.
